Discuss the scope of var, let, and const

1.Var is a function scope variable.It can be redeclared and reassigned

2.let is a block scope variable.It can not be redeclared but can be reassigned

3.const is a block scope variable.It can not be redeclared and reassigned

What do you mean by REST API?

REST stands for Representational State Transfer. This means that when a client requests a resource using a REST API, the server transfers back the current state of the resource in a standardized representation.REST API use following HTTP methods for operations:

  1. GET: for retrieving data.
  2. POST:for creating new data.
  3. PUT:for Updating existing data.
  4. DELETE:for removing data.
